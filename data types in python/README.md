# Python Data Types 📊

This folder contains a detailed exploration of built-in data types in Python and the concept of Type Casting.

### Files in this Folder:
* **datatype.py**: An introduction to core data types including Integers, Floats, Booleans, and Strings.
* **practicetypecasting.py**: Examples and exercises on converting one data type to another (e.g., String to Integer).
* **Questions.py**: A collection of logical problems and practice questions to test understanding of data types.

### Key Concepts Covered:
1. Identifying Data Types using the `type()` function.
2. Numeric types (int, float, complex).
3. Sequence and Mapping types.
4. Implicit and Explicit Type Casting (Type Conversion).
